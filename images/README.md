# Wedding Website Images

Folder ini berisi semua foto untuk website pernikahan Alya & Rafi.

## Struktur Folder

### `/hero`

**Foto Background untuk Hero Section**

- Nama file: `background.jpg` atau `background.png`
- Ukuran rekomendasi: Minimal 1920x1080px (Full HD)
- Format: JPG atau PNG
- Deskripsi: Foto besar untuk bagian hero (tampilan pertama)

### `/moments`

**Foto untuk "Our Journey" Section**

- Nama file sesuai urutan:
  - `moment-1.jpg` - Where it Began (October 2021)
  - `moment-2.jpg` - The First Date (November 2021)
  - `moment-3.jpg` - Wanderlust Hearts (Summer 2023)
  - `moment-4.jpg` - She Said Yes! (December 2025)
  - `nest-1.jpg` - Interior (Building Our Nest)
  - `nest-2.jpg` - Keys (Building Our Nest)

### `/gallery`

**Foto untuk Gallery Section**

- Nama file: `photo-1.jpg`, `photo-2.jpg`, dst
- Ukuran: Minimal 600x400px
- Format: JPG atau PNG

### `/couple`

**Foto Pasangan untuk Section Lainnya**

- Simpan foto utama pasangan di sini

## Cara Menggunakan

1. Tempatkan foto Anda di folder yang sesuai dengan keterangan di atas
2. Pastikan nama file sesuai dengan yang tercantum
3. Website akan otomatis menggunakan foto lokal ini

## Tips

- Gunakan format JPG untuk foto besar (lebih kecil file size)
- Gunakan format PNG untuk gambar dengan transparent
- Optimalkan size foto sebelum upload (gunakan tools seperti TinyPNG)
- Pastikan foto memiliki resolusi yang cukup untuk kualitas baik
